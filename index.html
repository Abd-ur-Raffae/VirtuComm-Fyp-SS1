VirtuComm

AI-Powered Communication Training Platform

Project Overview

VirtuComm is an innovative simulation platform that leverages AI and 3D models to help users—students, professionals, and job seekers—practice communication skills in real-world scenarios like job interviews, presentations, and academic discussions. With real-time feedback on tone, clarity, and posture, VirtuComm bridges the gap between theory and practice.

Key Features

  • Customizable scenarios: Interview, Podcast, Student-Teacher Interaction
  • AI-driven dialogues with real-time feedback
  • 3D avatars with lip-syncing and expressive animations
  • Real-time subtitles with customizable display
  • Resource recommendations for deeper learning
  • Secure user authentication

System Architecture

VirtuComm uses a client-server architecture:

  • Frontend: React, JavaScript, Three.js, Tailwind CSS
  • Backend: Django (Python 3.12)
  • APIs: Hugging Face Spaces, Rhubarb Lip Sync, ffmpeg
  • Data Flow: JSON and audio file exchanges

System Requirements

Minimum

  • CPU: Intel i5
  • RAM: 4GB/8GB
  • Graphics: Intel HD Graphics 520
  • Storage: 1.5GB

Recommended

  • CPU: Intel i5 or better
  • RAM: 8GB/16GB
  • Graphics: NVIDIA RTX 3060
  • Storage: 2GB

Supported OS: Windows 10/11. Browsers: Chrome, Firefox, Edge.
