VirtuComm

AI-Powered Communication Training Platform

Project Overview

VirtuComm is a cutting-edge simulation platform designed to enhance communication skills through AI-driven 3D scenarios. It allows users—students, professionals, and job seekers—to practice real-world interactions like job interviews, presentations, and academic discussions in a risk-free virtual environment. Powered by AI models (Zypher, Qwen), 3D rendering (Three.js), and lip-syncing (Rhubarb Lip Sync), VirtuComm provides real-time feedback on tone, clarity, and posture, fostering effective communication.

Key Features

  • Customizable scenarios: Interview, Podcast, Student-Teacher Interaction.
  • AI-generated dialogues with real-time feedback on voice modulation and body posture.
  • Text-to-speech and lip-syncing with 3D avatars for immersive experience.
  • Real-time subtitle generation with customizable display options.
  • Resource recommendation system for additional learning materials.
  • Secure user authentication and role-based access control.

System Architecture

VirtuComm operates on a client-server architecture:

  • Frontend: Built with React, JavaScript, Three.js, and Tailwind CSS for 3D rendering and user interface.
  • Backend: Powered by Django (Python 3.12) for AI processing and data management.
  • APIs: Hugging Face Spaces for dialogue generation, Rhubarb Lip Sync for animations, and ffmpeg for audio processing.
  • Data Flow: JSON and audio files exchanged between frontend and backend for seamless interaction.

  • Memory: 4GB/8GB RAM

Recommended

Supported OS: Windows 10/11. Browsers: Chrome, Firefox, Edge.
